# northpace-website

Public landing, privacy policy, and support pages for the **Northpace** iPhone + Apple Watch app, hosted on GitHub Pages.

## URLs
- Landing: https://astorms44.github.io/northpace-website/
- Privacy Policy: https://astorms44.github.io/northpace-website/privacy.html
- Support: https://astorms44.github.io/northpace-website/support.html

## Files
- `index.html` — landing page
- `privacy.html` — privacy policy (used in App Store Connect → App Privacy → Privacy Policy URL)
- `support.html` — support page (used in App Store Connect → Support URL)
- `style.css` — shared styles (light/dark)
